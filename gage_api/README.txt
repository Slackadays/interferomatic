GAGE PYTHON SDK FOR LINUX
-----------------------------------

Prerequisites:

  * Only versions of Python 3.6 and greater are supported.
  * The python development package (python-dev) and numpy must have been installed in the PC.


Build Gage Python library:

  Before running any Gage Python samples, you must build the Gage Python library:

     * Go to the folder "PyGage:
     * Run the following command line: 
	  python3 setup.py install

       Note:  on the command above, your python might be called "python" rather "python3".
